# Edusity

A modern educational website built using React and Vite.

## 🌐 Live Demo
https://edusity-rose-phi.vercel.app

## 📖 About
Edusity is a responsive educational website that provides information about courses, campus facilities, student testimonials, and academic opportunities.

## ✨ Features
- Responsive Design
- Modern User Interface
- Campus Gallery
- Student Testimonials
- Contact Section
- Smooth Navigation

## 🛠 Technologies Used
- React
- Vite
- JavaScript
- HTML5
- CSS3
